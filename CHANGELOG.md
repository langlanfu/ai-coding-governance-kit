# Changelog

All notable changes to this project will be documented in this file.

## [0.1.0] - 2026-05-03

### Added

- Initial local MVP of the AI Coding Team Governance Kit.
- README with project overview, quick start, principles, disclaimer, and Chinese summary.
- Tool-specific policies for GitHub Copilot, Codex-style agents, Claude Code, Cursor/AI IDEs, MCP tools, and browser automation.
- Sandbox checklist covering filesystem, network, credentials, shell, browser, MCP tools, GitHub tokens, CI/CD, databases, production, and log retention.
- Secure code review checklist, prompt/secret safety guidance, production change policy, incident response playbook, and team rollout guide.
- Templates for team AI policy, project AI risk assessment, PR review, sandbox audit, and agent run logs.
- Example policies for solo developers, startup teams, and agencies.
- MIT license, contributing guide, security policy, one-page checklist, and project status metadata.
